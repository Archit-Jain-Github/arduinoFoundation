# Led Example

```c
void setup()
{
    pinMode(13, OUTPUT); // Set pin 13 as an output pin for the LED
}
void loop()
{
    digitalWrite(13, HIGH); // Turn the LED on
    delay(1000);            // Wait for 1 second
    digitalWrite(13, LOW);  // Turn the LED off
    delay(1000);            // Wait for 1 second
}
```
